CSS CĂN BẢN

  1. CSS là gì?

    2. Trong lĩnh vực xây dựng, chúng ta có trang trí nội thất; trong lĩnh vực thẩm mỹ - làm đẹp, chúng ta có kỹ THUẬT makeup; trong lĩnh vực thiết kế web, chúng ta có CSS. CSS (Cascading Style Sheets) là một ngôn ngữ dùng để mô tả cách trình bày của tài liệu HTML. CSS giúp chúng ta kiểm soát bố cục của nhiều trang web cùng một lúc.

  2. Học CSS cần những gì?
    • Kiến thức cơ bản về HTML
    • Trình soạn thảo văn bản (Notepad, Notepad++, Sublime Text, VS Code,...)
    • Trình duyệt web (Google Chrome, Firefox, Cốc Cốc,...)
  3. Những lệnh thường gặp trong CSSL
    • color: Chỉ định màu cho chữ
    • font-size: Chỉ định kích thước chữ
    • font-family: Chỉ định kiểu chữ
    • text-align: Canh lề chữ (trái, phải, giữa)
    • background-color: Chỉ định màu nền
    • border: Tạo viền cho phần tử
